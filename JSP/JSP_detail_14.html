# Java ## 문제 발생 - 게시판 페이지를 슬라이스 하는 중 전체 게시글 배열 / 한 페이지 표시 수로 계산하는데 페이지 수 올림이 적용되지 않음. 블로그 제작떄도 같은 문제가 있었으나 블로그와 구조가 다르므로 적용 할 수가 없음. ## 문제 해결 - 정수 / 정수 = 정수.0 ex) 13/10 = 1.0 그래서 분자와 분모 모두 int가 아닌 double이여야 결과 값도 소수점을 포함하여 나옴. 나온 double 값은 올림하면 정상적으로 1이 아닌 2가 출력됨 확인 - [출처](https://okky.kr/article/73390)
